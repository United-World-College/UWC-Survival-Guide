**我們發布什麼。** 關於學業、大學申請、校園生活以及任何有助於學生適應 UWC 常熟生活的第一手指南。文章可以用英文、簡體中文或繁體中文撰寫。

**如何投稿。** 在貢獻者門戶註冊帳號，使用 Markdown 撰寫文章，然後提交審核。編輯會閱讀你的稿件，並決定直接發布、請求修改或提供反饋。如果你不熟悉 Markdown，可以先用一般文字寫好文章，再使用任意 LLM（如 ChatGPT 或 Claude）將其轉換為 Markdown 格式，然後將結果複製貼上到提交表單中。

**歡迎提議新分類。** 現有分類可能無法涵蓋所有值得撰寫的主題。如果你的文章不屬於任何現有分類，歡迎提議新增分類——請提前聯繫主編。

**有疑問？** 如果你對投稿流程、格式要求或其他事項有任何疑問，歡迎透過「關於」頁面聯繫主編。

**後續流程。** 通過審核的文章將以你的名義發布在網站上，並附有永久的作者主頁，匯集你的所有作品。你的名字也將出現在本頁的貢獻者列表中。
