在某些社群媒體上的所謂社會菁英敘事裡，最優雅的莫過於「一邊寫程式、一邊讀莎士比亞」。就是在深夜的咖啡館裡，左手是滿螢幕跳動的 CUDA 核心函式，右手邊恰到好處地擺著一本翻開的《哈姆雷特》的那種。這也恰好是某些菁英教育流水線最篤信的配方。這種「通才」的人設極具誘惑力，因為它精準地切中了一種古老的渴望：用一個足夠浪漫、足夠深沉的自我敘事，來宣告自己屬於某一類人。莎士比亞和程式的組合只是恰好提供了這套敘事，因為它暗示你既有邏輯的硬度，又有靈魂的厚度。但換一個時代、換一個圈子，符號會變，這份渴望不會變。這種 class signaling 邏輯的構建通常分兩步：第一步，用技術的深度來彰顯自己與普通人不在同一個智識層次。畢竟，能真正掌握這些東西的人本就不多；第二步，在此之上再披一層浪漫與深沉，來表明自己不只是一個聰明的執行者，而是一個有靈魂的人。當然，也有人懶得走第一步，直接從第二步開始。對他們來說技術不重要，重要的是那份與生俱來的精神氣質。普通工程師則是這套敘事裡隱含的對照組：他們也懂技術，但止步於此，所以終究只是工具。而智識與靈魂兼得的菁英，才是真正的稀缺。

*註：這裡要談的不是那些真心熱愛兩者的人，而是把博雅當 class signaling 的人。*

為什麼很多人執著於莎士比亞呢？我認為這不是因為他們真的被那些文字擊中，而是因為「讀莎士比亞的人」這個形象本身就自帶一種難以被定義的、凌駕於實用之上的貴族氣質。（當然這個符號可以是任何東西——欣賞文藝作品、學習量子力學、鑽研抽象代數，甚至換成品酒、古典樂、禪修都行，只要它不實用又足夠有門檻。）但這種對博雅的消費，恰恰是對博雅最大的誤解。真正的莎士比亞並不溫情，他展現的是血淋淋的權力鬥爭與人性困境，讀進去的人往往不是變得更浪漫，而是變得更不安。在這裡，博雅成了兩撥人共同的提線木偶：有的技術人用它來裝點技術身份的單調，有的文藝人用它來填充思想空洞的門面。他們一起構建了一個荒誕的共識：博雅是一種可以脫離嚴謹邏輯而獨立存在的「氛圍感」，是一種可以被擺出來的姿態，而不是一個需要被真正進入的領域。就像許多熱衷談論永續發展的人，對碳中和的口號倒背如流，卻說不清楚一個最基本的能源系統是怎麼運作的。符號被消費了，但符號背後的東西從未被觸碰。

Class signaling 最令人沉溺的幻覺莫過於此：當一個人用深沉或浪漫的標籤裝點自己時，他不是覺得自己在表演，而是他覺得自己就是那種技術和靈魂都有的人。這種 class signaling 最隱秘的毒性就在於它將博雅精神工具化，結果往往是既消解了技術的硬度，也抽空了人文的厚度。要走出這種表演，出口其實並不在變得更深沉那一側，而在不再表演這一側。這裡還有個容易混淆的概念——平庸。平庸和不表演看上去像，其實完全不一樣：一個人可以安靜地深入一件事，不需要觀眾；也可以平平淡淡地度過四年、甚至是一輩子，把寫程式或寫文章只當成謀生手段。前者是不表演的深入，後者是不表演的普通，實際上它們都是在沒有鏡頭的地方做出自己的選擇。假如想把日子過得平淡一點，坦然承認自己既讀不懂程式的邏輯，也讀不進莎士比亞的十四行詩，本身就是一種完整的誠實，它不需要和誰比較，也不需要一個更高的位置來證明自己。

我認為 class signaling 的核心，是把對思想深度的幻想當成深度本身。而這種替換之所以能成立，是因為它往往發生在領域之外。外面的人看不見領域內部的具體紋理，只看得見一個輪廓，於是一個關於自己是哪種人的想像就誕生了。這個問題其實也不是自詡博雅的人獨佔的，學術圈裡也有一個同構的版本，我曾經在網路上就看到德州農工的一位教授辛辣地評價過：

> 有些人鄙夷那些高產的灌水者，自己弄了幾年，總以為憋大招能出一個[何愷明](https://en.wikipedia.org/wiki/Kaiming_He)或[李飛飛](https://en.wikipedia.org/wiki/Fei-Fei_Li)。結果東西磨出來後發現，還不如人家大實驗室順手灌水的成果。

實際上鄙夷灌水的人和自詡博雅的人共享的是同一種優越感，他們表面上在做截然不同的事：一個在談產出質量，一個在談身份氣質，但在更深的層次上，他們都在用一個關於「我是哪種人」的想像，替代真正的產出。憋大招的人用「我在做大事」的自我形象，逃避自己遲遲沒有成果的焦慮；自詡博雅的人用「我有靈魂」的自我形象，逃避自己在任何一個領域都沒真正扎進去的事實。兩者其實都是把敘事當成結果，把姿態當成行動。真正十年磨一劍的人，永遠是那些敢於挑戰大問題、敢在無人區硬碰硬的人，而不是那些躲在長期主義的幌子下自我感動的人。

我一直都認為弄清楚一個漢字的古代讀音，其實與發現一顆小行星是同樣重要的。任何領域的尊嚴，都只產生於 down-to-earth 地解決複雜問題的陣痛之中。很多這樣 down-to-earth 的畫面不上鏡，也不適合在咖啡館擺拍：一個人被一個具體的、難纏的問題拽住了，然後他選擇不鬆手。這種不鬆手是有尊嚴的，而這種尊嚴在所有領域裡是等價的。把學科分成高低的，從來不是做事的人，而是看熱鬧的人。真正值得期待的，是一個人在推敲文字的張力時，能像他在壓榨程式碼的效能時一樣，始終帶著對底層邏輯的敬畏，去追尋那種純粹的、不被學科邊界所定義的興奮。而如果一個人發現自己對兩者都提不起興趣，那也沒什麼，因為對很多我們這個年紀的人而言能坦然接受自己的平凡往往需要更大的勇氣。畢竟，真正的優雅從來不是活在某種精心構建的人設之中，而是永遠願意去擁抱那個不完美但真實的自己。

## 附錄：一個關於誠實的自檢清單

在 UWC 這種期待值極高的環境下，我們很難不被某種完美菁英的敘事裹挾。如果你也曾懷疑過自己是否在進行某種 Class Signaling，不妨在深夜獨處時，試著溫柔地問自己幾個問題：

**關於觀眾：**
如果在你的生活裡，沒有任何人會知道你在讀什麼、在學什麼，甚至沒有人會因為你的這種博雅而給你加分，你是否依然會被那些文字或邏輯本身所打動？如果那種純粹的喜悅消失了，或許是因為你背負了太沉重的、演給別人看的劇本。

**關於優越感：**
你是否在潛意識裡，把這種博雅的身份當成了一種防禦機制？比如，當你在某個領域遇到瓶頸時，會透過「我還有另一種深度」來消解當下的挫敗。真正的底蘊不應該是用來俯視他人的台階，而應該是支撐你走過低谷的底氣。

**關於避難所：**
你是在消費某種符號，還是在體驗它背後的精神？無論是躲進書本還是躲進程式，承認自己累了、需要一個喘息的空間應該被鼓勵。我們需要區分的是：你是在尋找一個讓自己活下去的避難所，還是在修築一座用來彰顯不同的護城河？

**關於笨拙的誠實：**
你是否允許自己的不優雅？對一個領域真正的探索在初期往往都是狼狽的。如果你因為害怕破壞某種人設而不敢承認自己在某個領域其實只是個笨拙的初學者，那麼這種人設本身就是你成長的枷鎖。

**關於等級感：**
你在談論自己做的事情時，腦子裡是否有一張隱形的學科地圖，在裡面某些領域配得上被提及，另一些上不了檯面？如果有，問問自己：這張地圖是你在某個領域已經深入到底之後畫出來的，還是你還沒真正下場時就已經默認的前設？前者是經驗，後者只是你替自己省下的那段路。

**寫在最後：**

承認自己偶爾需要人設並不羞恥，這甚至是社交生活的一種潤滑劑。但請警惕，別讓這些訊號成為了你與真實世界、真實自我之間的圍牆。在這個人人都想成為完美菁英敘事下的主角的時代，我更想致敬的是那些敢於承認自己沒讀懂莎士比亞的勇氣，或坦然面對程式邏輯平庸的誠實。因為我們永遠不需要用另一種高度來掩蓋最真實的自己。
