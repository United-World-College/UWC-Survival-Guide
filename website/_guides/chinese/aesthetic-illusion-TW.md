在某些社群媒體上的所謂社會菁英敘事裡，最優雅的莫過於「一邊寫程式、一邊讀莎士比亞」。就是在深夜的咖啡館裡，左手是滿螢幕跳動的 CUDA 核心函式，右手邊恰到好處地擺著一本翻開的《哈姆雷特》的那種。這也恰好是某些菁英教育流水線最篤信的配方。這種「通才」的人設極具誘惑力，因為它精準地切中了人們對階級審美和浪漫敘事的嚮往。但如果剝開這層優雅的濾鏡，你會發現這種敘事往往只是一種極其危險的審美代償。

*註：這裡要談的不是那些真心熱愛兩者的人，而是把博雅當 class signaling 的人。*

為什麼很多人執著於莎士比亞呢？因為當一個人在處理系統死結、優化 CUDA 核心函式感到智力窒息時，把人文底蘊當作舒適區是最體面的退路。但這其實是一種錯覺——真正的莎士比亞並不溫情，他展現的是血淋淋的權力鬥爭與人性困境，其解構難度並不亞於任何複雜的分散式系統。所謂躲進人文，其實只是躲進了關於人文的幻象裡。在這裡，人文成了兩撥人的共享贓物：技術人用它來逃避邏輯的挫敗，文青用它來掩蓋認知的空洞。前者覺得人文軟，所以可以隨手拿來墊腳；後者覺得人文虛，所以可以用來當遮羞布。他們共同構建了一個荒誕的共識：即人文是可以脫離嚴謹邏輯而獨立存在的「氛圍感」。但這恰恰是對人文最大的誤解，因為沒有邏輯支撐的情感永遠只是廉價的煽情，沒有深刻認知的文藝永遠只是無病呻吟。實際上那些只堆砌程式邏輯、卻拒絕思考工作意義的程式機器，與那些只會感嘆風月、卻拒絕理解世界運行底層邏輯的文青，本質上是同一種形態的逃避。

Class signaling 最令人沉溺的幻覺莫過於此：「我程式寫得不如 [ICPC](https://en.wikipedia.org/wiki/International_Collegiate_Programming_Contest) 戰神，或者我根本就不屑於弄懂這些枯燥的邏輯，但我有更深厚的人文關懷。所以就算這些人現在在 ICPC 上屠榜，以後也得給我打工。」這種 class signaling 最隱秘的毒性就在於它將人文精神工具化，結果往往是既消解了技術的硬度，也抽空了人文的厚度。這不僅是對兩者的雙重損耗，更是一種智力上的自我粉飾。其實，擁抱平庸是每個人的權利。選擇平平淡淡地度過四年、甚至是一輩子，或者承認自己只是將編程視為謀生手段，這是一種職業上的誠實，也是對自己的坦誠。事實上，承認自己既讀不懂程式的邏輯，也讀不進莎士比亞的十四行詩，甚至比這種以此代彼的優越感要高尚得多。

我其實一直覺得這種 class signaling 的心理，與在學術研究中有些人對灌水的盲目鄙夷與對憋大招的病態執著極為相似。我曾經在網路上看到德州農工大學的一位教授對這種心理有過一段辛辣的評價：

> 有些人鄙夷那些高產的灌水者，自己弄了幾年，總以為憋大招能出一個[何愷明](https://en.wikipedia.org/wiki/Kaiming_He)或[李飛飛](https://en.wikipedia.org/wiki/Fei-Fei_Li)。結果東西磨出來後發現，還不如人家大實驗室順手灌水的成果。

這正是對學術虛榮的滑稽之處的諷刺。真正十年磨一劍的人，永遠是那些敢於挑戰大問題、在無人區硬碰硬的人，而不是那些躲在長期主義的幌子下自我感動的人。這對自詡「一邊寫程式、一邊讀莎士比亞」的人也是同理。當一個人試圖用人文學科的格調來稀釋技術焦慮時，他其實同時消解了兩個領域的嚴肅性。最終，他既沒能在程式的底層邏輯裡獲得力量，也沒能在莎士比亞的悲劇裡獲得救贖。這種把博雅當 class signaling 的迷戀，最終只是讓人在邏輯與審美的雙重退縮中，把自己活成了一場滑稽的悲劇。

我一直都認為弄清楚一個文字的古代讀音，其實與發現一顆小行星是同樣重要的。這是因為任何領域的尊嚴，都只產生於解決複雜問題的陣痛之中。比起附庸風雅的掛靠，我更希望看到一個人在推敲文字的張力時，能像他在壓榨程式碼的效能時一樣，始終帶著對底層邏輯的敬畏，去追尋那種純粹的、不被學科邊界所定義的真實。畢竟，真正的優雅從非逃避複雜的藉口，而是征服複雜後的純粹。

## 附錄：一個關於誠實的自檢清單

在 UWC 這種期待值極高的環境下，我們很難不被某種「完美菁英」的敘事裹挾。如果你也曾懷疑過自己是否在進行某種 Class Signaling，不妨在深夜獨處時，試著溫柔地問自己幾個問題：

**關於觀眾：**
如果在你的生活裡，沒有任何人會知道你在讀什麼、在學什麼，甚至沒有人會因為你的這種「博雅」而給你加分，你是否依然會被那些文字或邏輯本身所打動？如果那種純粹的喜悅消失了，或許是因為你背負了太沉重的、演給別人看的劇本。

**關於優越感：**
你是否在潛意識裡，把這種跨學科的身份當成了一種防禦機制？比如，當你在某個領域遇到瓶頸時，會透過「我還有另一種深度」來消解當下的挫敗。真正的底蘊不應該是用來俯視他人的台階，而應該是支撐你走過低谷的底氣。

**關於避難所：**
你是在消費某種符號，還是在體驗它背後的精神？無論是躲進書本還是躲進程式，承認自己累了、需要一個喘息的空間應該被鼓勵。我們需要區分的是：你是在尋找一個讓自己活下去的避難所，還是在修築一座用來彰顯不同的護城河？

**關於笨拙的誠實：**
你是否允許自己在跨界時表現得「不優雅」？真正的跨學科探索在初期往往都是狼狽的。如果你因為害怕破壞「文理雙修」的人設而不敢承認自己在某個領域其實只是個笨拙的初學者，那麼這種人設本身就是你成長的枷鎖。

**寫在最後：**

承認自己偶爾需要「人設」並不羞恥，這甚至是社交生活的一種潤滑劑。但請警惕，別讓這些訊號成為了你與真實世界、真實自我之間的圍牆。寫這篇文章並不是為了批判那些在寫程式間隙翻開書頁的人。恰恰相反，在這個人人都想成為完美菁英敘事下的主角的時代，我更想致敬的是那些敢於承認自己沒讀懂莎士比亞的勇氣，或坦然面對程式邏輯平庸的誠實。因為我們永遠不需要用另一種高度來掩蓋當下的狼狽。

最好的優雅，是當你終於不需要向任何人證明自己的時候，依然對這個世界的運行——無論是那行難搞的程式碼，還是那段晦澀的詩句——保有那一絲純粹的好奇與敬畏。
