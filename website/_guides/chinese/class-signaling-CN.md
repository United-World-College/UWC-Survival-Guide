在有些社交媒体上的所谓社会精英叙事里，最优雅的莫过于"一边写代码、一边读莎士比亚"。就是在深夜的咖啡馆里，左手是满屏跳动的 CUDA 核函数，右手边恰到好处地摆着一本翻开的《哈姆雷特》的那种。这也恰好是某些精英教育流水线最笃信的配方。这种"通才"的人设极具诱惑力，因为它精准地切中了一种古老的渴望：用一个足够浪漫、足够深沉的自我叙事，来宣告自己属于某一类人。莎士比亚和代码的组合只是恰好提供了这套叙事，因为它暗示你既有逻辑的硬度，又有灵魂的厚度。但换一个时代、换一个圈子，符号会变，这份渴望不会变。这种 class signaling 逻辑的构建通常分两步：第一步，用技术的深度来彰显自己与普通人不在同一个智识层次。毕竟，能真正掌握这些东西的人本就不多；第二步，在此之上再披一层浪漫与深沉，来表明自己不只是一个聪明的执行者，而是一个有灵魂的人。当然，也有人懒得走第一步，直接从第二步开始。对他们来说技术不重要，重要的是那份与生俱来的精神气质。普通程序员则是这套叙事里隐含的对照组：他们也懂技术，但止步于此，所以终究只是工具。而智识与灵魂兼得的精英，才是真正的稀缺。

*注：这里要谈的不是那些真心热爱两者的人，而是把博雅当 class signaling 的人。*

为什么很多人执着于莎士比亚呢？我认为这不是因为他们真的被那些文字击中，而是因为"读莎士比亚的人"这个形象本身就自带一种难以被定义的、凌驾于实用之上的贵族气质。（当然这个符号也可以是任何其他的文艺作品。）但这种对人文的消费，恰恰是对博雅最大的误解。真正的莎士比亚并不温情，他展现的是血淋淋的权力斗争和人性困境，读进去的人往往不是变得更浪漫，而是变得更不安。在这里，人文成了两拨人共同的提线木偶：有的技术人用它来装点技术身份的单调，有的文艺人用它来填充思想空洞的门面。他们一起构建了一个荒诞的共识：博雅是一种可以脱离严谨逻辑而独立存在的"氛围感"，是一种可以被摆出来的姿态，而不是一个需要被真正进入的领域。就像许多热衷谈论可持续发展的人，对碳中和的口号倒背如流，却说不清楚一个最基本的能源系统是怎么运行的。符号被消费了，但符号背后的东西从未被触碰。

Class signaling 最令人沉溺的幻觉莫过于此：当一个人用深沉或浪漫的标签装点自己时，他不是在掩盖什么，而是真正地相信这个形象就是自己。他不觉得自己在表演，他觉得自己就是那种技术和灵魂都有的人。这种 class signaling 最隐秘的毒性就在于它将人文精神工具化，结果往往是既消解了技术的硬度，也抽空了人文的厚度。其实，拥抱平庸是每个人的权利。选择平平淡淡地度过四年、甚至是一辈子，或者承认自己只是将某件事视为谋生手段，是对职业的诚实，也是对自己的诚实。坦然接受自己既读不懂代码的逻辑，也读不进莎士比亚的十四行诗，事实上要比这种用形象掩盖现实的优越感要高尚得多。

我其实一直觉得这种 class signaling 心理的核心，是把对深度的幻想，当成了深度本身。我曾经在网上看到德州农工的一位教授就辛辣地评价过学术圈里的同类现象：

> 有些人鄙视那些高产的灌水者，自己弄了几年，总以为憋大招能出一个[何恺明](https://en.wikipedia.org/wiki/Kaiming_He)或[李飞飞](https://en.wikipedia.org/wiki/Fei-Fei_Li)。结果东西磨出来后发现，还不如人家很多大实验室顺手灌水的成果。

实际上鄙视灌水的人和自诩博雅的人共享的是同一种优越感：都在用一个关于自己是哪种人的想象，居高临下地俯视那些只是踏实做事的人。真正在学术界十年磨一剑的人，永远是那些敢挑战大问题、敢在无人区硬碰硬的人，而不是躲在长期主义幌子下自我感动的人。把博雅当精英符号，不过是同一种虚荣的另一个版本。

我一直都认为弄清楚一个文字的古代读音，其实与发现一颗小行星是同样重要的。这是因为任何领域的尊严，都只产生于解决复杂问题的阵痛之中。比起附庸风雅的挂靠，我更希望看到一个人在推敲文字的张力时，能像他在压榨代码的效能时一样，始终带着对底层逻辑的敬畏，去追寻那种纯粹的、不被学科边界所定义的真实。而如果发现自己对两者都提不起兴趣，那也没什么，因为坦然接受自己的平凡往往需要更大的勇气。毕竟，真正的优雅从来不是活在某种精心构建的人设之中，而是永远愿意去拥抱那个不完美但真实的自己。

## 附录：一个关于诚实的自检清单

在 UWC 这种期待值极高的环境下，我们很难不被某种“完美精英”的叙事裹挟。如果你也曾怀疑过自己是否在进行某种 Class Signaling，不妨在深夜独处时，试着温柔地问自己几个问题：

**关于观众：**
如果在你的生活里，没有任何人会知道你在读什么、在学什么，甚至没有人会因为你的这种“博雅”而给你加分，你是否依然会被那些文字或逻辑本身所打动？如果那种纯粹的喜悦消失了，或许是因为你背负了太沉重的、演给别人看的剧本。

**关于优越感：**
你是否在潜意识里，把这种跨学科的身份当成了一种防御机制？比如，当你在某个领域遇到瓶颈时，会通过“我还有另一种深度”来消解当下的挫败。真正的底蕴不应该是用来俯视他人的台阶，而应该是支撑你走过低谷的底气。

**关于避难所：**
你是在消费某种符号，还是在体验它背后的精神？无论是躲进书本还是躲进代码，承认自己累了、需要一个喘息的空间应该被鼓励。我们需要区分的是：你是在寻找一个让自己活下去的避难所，还是在修筑一座用来彰显不同的护城河？

**关于笨拙的诚实：**
你是否允许自己的不优雅？对一个领域真正的探索在初期往往都是狼狈的。如果你因为害怕破坏某种人设而不敢承认自己在某个领域其实只是个笨拙的初学者，那么这种人设本身就是你成长的枷锁。

**写在最后：**

承认自己偶尔需要人设并不羞耻，这甚至是社交生活的一种润滑剂。但请警惕，别让这些信号成为了你与真实世界、真实自我之间的围墙。写这篇文章并不是为了批判那些在写代码间隙翻开书页的人。恰恰相反，在这个人人都想成为完美精英叙事下的主角的时代，我更想致敬的是那些敢于承认自己没读懂莎士比亚的勇气，或坦然面对代码逻辑平庸的诚实。因为我们永远不需要用另一种高度来掩盖最真实的自己。
